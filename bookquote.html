개인정보처리방침

앱 이름: 책문장
패키지명: com.nomalab.book_quote_saver
시행일: 2026-05-26

본 앱(이하 "앱")은 사용자가 책에서 발췌한 좋은 문장을 촬영·OCR로 추출해 기기 내에서 수집·보관하는 독서 인용 앱입니다. 본 방침은 앱이 어떤 정보를 처리하고, 어떻게 보관하며, 누구와 공유하는지 설명합니다.

1. 수집하는 정보

정보용도보관
사용자가 입력·저장한 책 정보 (제목·저자·ISBN·표지 URL) 책장 표시 및 문장과 연결 사용자 기기 내부 저장소, 외부 송신 없음
인용 문장 본문·페이지·메모 사용자가 직접 저장한 인용 보관 사용자 기기 내부 저장소, 외부 송신 없음
책 페이지 촬영 사진 OCR 텍스트 추출 및 인용 원본 보존 사용자 기기 내부 저장소, 외부 송신 없음
ISBN (바코드 인식 시) 카카오 책 검색 API로 책 정보 자동 조회 카카오에 ISBN만 전송, 개인 식별 정보 미전송
광고 식별자 (AAID) AdMob 광고 송출 Google AdMob 정책에 따름
결제 정보
(프리미엄 구매 시)		 Google Play 결제 시스템 Google Play 정책에 따름. 앱은 결제 정보 자체를 저장·접근하지 않음

본 앱은 자체 서버를 운영하지 않으며, 사용자의 책·문장·사진 데이터를 외부 서버로 전송·저장하지 않습니다.

2. 제3자 제공 / 처리위탁

  • 카카오 (책 검색 API): ISBN을 통해 책 제목·저자·표지를 조회 — 카카오 정책 보기
  • Google AdMob: 광고 송출 및 광고 식별자 처리 — 정책 보기
  • Google Play 결제 (Google Play Billing): 프리미엄(광고 제거·공유 푸터 제거) 일회성 결제 처리. 자동갱신 구독 없음.
  • Google ML Kit: 텍스트 인식(OCR)·바코드 인식. 인식은 전부 기기 내에서 처리되며 외부로 전송되지 않음.

3. 사용자 권리

  • 저장한 책·문장·사진은 앱 내에서 언제든 직접 삭제할 수 있습니다.
  • 광고 식별자 초기화: 안드로이드 설정 → 개인정보 보호 → 광고 → "광고 ID 재설정"
  • 앱 삭제 시 기기 내 저장된 모든 데이터가 함께 삭제됩니다.
  • 데이터·결제 관련 문의: 아래 이메일

4. 카메라 / 저장소 권한

  • 카메라: 책 뒤표지 바코드와 페이지 문장 촬영. 거부 시 사진 기반 기능은 사용할 수 없습니다.
  • 사진/미디어 접근: 갤러리에서 페이지 사진을 가져올 때 사용.
  • 인터넷: 카카오 책 검색 API, 표지 이미지 다운로드, 광고 송출, 결제 통신.

5. 보관 기간 및 파기

사용자가 앱 내에서 책·문장을 삭제하거나 앱을 제거할 때까지 사용자 기기에 보관됩니다. 본 앱 운영자는 사용자 데이터를 보관하는 별도 서버를 운영하지 않으므로, 별도의 파기 절차가 발생하지 않습니다.

6. 13세 미만 아동의 정보

본 앱은 13세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

7. 방침 변경

법령 또는 서비스 변경에 따라 본 방침이 수정되는 경우, 본 페이지를 통해 공지합니다.
